# Documentación Técnica - CRM con Clientes y Notas

## Información General

**Aplicación:** CRM (Customer Relationship Management) con gestión de clientes y notas  
**Stack Tecnológico:** HTML + JavaScript  
**Repositorio:** https://github.com/luisernestosalas/app-og0n5  
**URL de Producción:** https://app-og0n5-cw1fa0d11-luisernestosalas-2775s-projects.vercel.app  
**Plataforma de Despliegue:** Vercel  

## Descripción del Sistema

Sistema CRM básico desarrollado con tecnologías web frontend que permite la gestión de clientes y sus respectivas notas asociadas. La aplicación utiliza un enfoque client-side con almacenamiento local.

## Arquitectura Técnica

### Frontend
- **HTML5:** Estructura y marcado semántico
- **JavaScript (Vanilla):** Lógica de negocio y manipulación del DOM
- **CSS:** Estilos y presentación (implícito en el stack)

### Características Principales
- Interfaz de usuario responsive
- Gestión completa de clientes (CRUD)
- Sistema de notas asociadas a clientes
- Almacenamiento persistente local
- Despliegue automatizado en Vercel

### Gestión de Clientes
