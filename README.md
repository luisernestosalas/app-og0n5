# CRM App - Gestión de Clientes y Notas

Una aplicación web sencilla para la gestión de clientes y notas, desarrollada con tecnologías web estándar.

## 🚀 Demo en Vivo

[Ver aplicación](https://app-og0n5-cw1fa0d11-luisernestosalas-2775s-projects.vercel.app)

## 📋 Características

- **Gestión de Clientes**: Crear, editar, visualizar y eliminar clientes
- **Sistema de Notas**: Asociar notas a cada cliente para seguimiento
- **Interfaz Responsiva**: Diseño adaptable para dispositivos móviles y desktop
- **Almacenamiento Local**: Los datos se guardan en el navegador (localStorage)
- **Búsqueda y Filtros**: Encuentra clientes rápidamente

## 🛠️ Stack Tecnológico

- **HTML5** - Estructura y marcado
- **CSS3** - Estilos y diseño responsivo
- **JavaScript ES6+** - Lógica de la aplicación y manipulación del DOM
- **LocalStorage** - Persistencia de datos en el cliente
